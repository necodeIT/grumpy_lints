# modular_foundation_lints

Linting rules for using modular_foundation

## Usage

To use this package, add it to your `dev_dependencies` in `pubspec.yaml`:

```yaml
dev_dependencies:
  modular_foundation_lints: ^0.1.0
```

Then, create or update your `analysis_options.yaml` to include these rules:

```yaml
include: package:modular_foundation_lints/analysis_options.yaml
```

## Features

This package provides a comprehensive set of lint rules to ensure consistent code quality and style across modular_foundation projects, including:

- Error detection rules to catch common mistakes
- Style rules for consistent code formatting
- Best practices enforcement
- Flutter-specific rules for widget development
